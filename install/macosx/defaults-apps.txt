# Dominios de preferencias (defaults) a sincronizar entre Macs.
# Reemplaza a mackup. Un dominio por línea; '#' inicia comentario.
#
# Cómo encontrar el dominio de una app:
#   defaults domains | tr ',' '\n' | grep -i <nombre>
#   osascript -e 'id of app "Nombre de la App"'
#
#   backup-defaults   -> exporta estos dominios a la carpeta de backup (Dropbox)
#   restore-defaults  -> los importa (install.sh lo hace en un Mac nuevo)
